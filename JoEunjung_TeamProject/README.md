## 포트폴리오 제출 참고

작업일지와 수행 계획서는 GitHub에서 바로 확인하기 쉽도록 PDF 파일을 함께 포함했습니다.
추후 수정이 가능하도록 PPTX, DOCX 원본 파일도 함께 보관했습니다.

개인 포트폴리오 PPTX는 수정이 끝난 뒤 최신본으로 다시 교체할 예정입니다.
