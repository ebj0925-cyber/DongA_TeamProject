## 담당 역할

- C구간에 필요한 기사, 기관, 공공 지원사업, 논문 자료 조사
- 도시 정보 레이어와 지원기관 안내에 활용할 C구간 자료 정리
- 개인 작업일지 및 팀 작업일지 작성과 문서화
- 발표자료와 개인 포트폴리오 구성 정리
- 프로젝트 진행 과정, 피드백, 산출물, 향후 보완 사항 기록
