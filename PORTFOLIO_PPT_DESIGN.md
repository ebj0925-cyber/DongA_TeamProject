# Planning Portfolio PPT Design Guide

## Concept
- Role positioning: 기획, 사용자 경험 흐름, 스토리보드, 연출, 기능 구조 설계
- Visual tone: white canvas, light blue panels, navy typography, clean technical portfolio
- Message: 추상적인 고립감을 설문, 3D 체험, 결과, 통계, 메시지 기능으로 연결한 경험 설계

## Slide System
- Size: 16:9 widescreen
- Background: #F7FAFF
- Primary text: #071B4D
- Accent blue: #245BDB
- Soft panel: #EEF4FF
- Border: #C9D7F2
- Body text: #17213A

## Reusable Components
- Section badge: top-left navy square with white two-digit number
- Header: page number + title + small project label
- Main title: large navy text, blue emphasis via separate text box
- Feature card: light panel, thin blue border, icon circle, title, two-line body
- Flow arrow: blue right arrow between cards
- Keyword strip: long rounded panel with five compact keywords
- Goal box: bottom rounded panel with icon block and one-sentence result

## Content Rules
- One slide has one central claim.
- Each card title is short, usually 4 to 8 Korean words.
- Body text stays under two lines.
- Functional slides show both planning intent and implemented system.
- Use project screenshots only as evidence, not decoration.
